Etiquetas semánticas

HTML Semantico es el uso de las etiquetas HTML para reforzar la semántica de la información en las paginas Web mas que simplemente redefinir su forma de presentación, una etiqueta se califica como semantica si tiene que ver el significado, es decir, si nos informa sobre lo que trata su contenido.

*Celaya Luna, A. (2014). Creación de páginas web: HTML5. p. 1 - 60.https://elibro-net.bibliotecavirtual.unad.edu.co/es/ereader/unad/56045

CSS grid

grid es uno de los nuevos valores de HTML 5 que podemos dar a la propiedad de un display de un elemento y que nos permite maquetar las paginas web de una manera mucho mas facil que se hacia con la forma tradicional. Ahora grid es un estandar y no es necesario recargar ni ensuciar el codigo para aplicarlo. Todo se aplica desde el CSS, por medio de nuevos atributos, valores y unidades con una variada sintaxis.Al ser una especificación estable, estandar y soportada por practicamente todos los navegadores modernos, se considera CSS, por lo que se puede utilizar en todos los sitios donde ya se utilice un codigo CSS: sitios estaticos, programación con un lenguaje determinado

Attardi, J. (2020). CSS Grid. In Modern CSS (pp. 255-280). Apress, Berkeley, CA.https://link.springer.com/chapter/10.1007/978-1-4842-6294-8_12
